Si vous voyez ce texte, c'est que cette page a besoin d'informations compémentaires de votre part pour fonctionner.

Il faut que le dossier dans lequel est ce fichier contienne également un fichier nommé _icescrum_api.js dans lequel vous devez mettre:

  1. Votre token d'accès à l'API Icescrum. var iceScrumToken = '91030e0ea9a9ab58...'

    Vous générez ou retrouvez votre token dans votre profil IceScrum ("My Account") sous l'onglet 'API token'

  2. Le code Icescrum de votre projet: projectId = 'XXXXXX'

    Vous trouverez ce code dans l'url quand vous êtes sur votre projet dans Icescrum (https://icescrum.cpnv.ch/p/XXXXXX/#/project)

Veillez également à ce que le fichier _icescrum_api.js soit exclu du système de versionning, si vous ne voulez pas que votre token se retrouve sur Github!
